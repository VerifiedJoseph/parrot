ü¶ú About

Parrot is a browser-based program for viewing tweets from zip archives created with the Twitter Media Downloader browser extension.

üìñ How to

Select a zip archive created with the Twitter Media Downloader by clicking the 'browse' button above and the viewer will generate a twitter style interface for viewing the tweets.

‚ùì Q & A

Is any data upload from my device?
No. The viewer runs completely within your web browser and does not collect or transmit any data.

Can I run it offline?
Yes. The viewer can be run offline and a download is available.

Does it use any third party JavaScript libraries?
Yes. The viewer uses three third party JavaScript libraries:

  • JSZip (v3.10.1) - JavaScript class for generating and reading zip files.
  • Papa Parse (v5.4.1) - Fastest in-browser CSV parser for JavaScript.
  • Autolinker.js (v4.0.0) - Utility to automatically link URLs.
